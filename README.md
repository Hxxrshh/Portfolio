# Harsh Soni Portfolio

A modern, responsive personal portfolio website built using HTML, CSS, and JavaScript.

## Highlights
- Light premium design with subtle blue accents
- Fully responsive layout for desktop and mobile
- Smooth scrolling navigation and reveal animations
- Dedicated sections for hero, about, leadership, skills, projects, achievements, career focus, and contact
- Contact form UI with basic front-end interaction

## Run Locally
Open `index.html` directly in a browser, or serve locally:

```bash
python3 -m http.server 5500
```

Then open `http://localhost:5500`.
